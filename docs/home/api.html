关于软件
本软件为微信助手类软件
站在微信的肩膀上提供生产生活效率
安全无后门,无后台操作,所有操作由用户或插件发起
软件不限制微信版本
您可在任意时候开启或卸载功能
当本机已有微信登录,可直接启用功能
也可以新开启一个微信,不会对您当前微信做任何操作
本社区将用户体验放在第一位
希望开发者也以用户为中心,己所不欲勿施于人

为了稳定性,采用window服务的方式部署,可一键安装卸载
软件内嵌web浏览器,开启后会打开首页,您也可以直接关闭软件,使用其他浏览器操作
界面采用html设计,插件功能采用JScript脚本开发
您可以任意时候打开浏览器在线修改您的插件
